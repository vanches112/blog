Установка Minecraft и Python, запуск сервера

Как установить Minecraft

Рассмотрим установку игры Minecraft, которая позволит не только проводить время с удовольствием, но и с пользой, так как мы с вами научимся программировать в Minecraft на языке Python.

Установка minecraft

Для начала необходимо перейти на сайт https://tlauncher.org/

Нажать на кнопку скачать Tlaucher

Здесь необходимо выбрать операционную систему Вашего компьютера.

Запускаем скаченный файл

Ставим необходимые галочки и завершаем установку

Если у Вас раньше не было установлено програмное обеспечение Java, то лаунчер запросит разрешение на его установку.

  • Введите желаемое имя
  • Выберите версию и нажмите на кнопку Установить

Перейдем к установке среды для программирования на языке Python

Перейдите на сайт https://www.python.org/downloads/

Выбор необходимой системы

Выберите на сайте необходимую операционную систему и скачайте Python.

Установка

Запускаем скаченный файл и производим установку

Как только все будет готово, появится сообщение

Снимок экрана 2022-05-29 в 15.33.18

После установки запускаем среду программирования IDLE

Снимок экрана 2022-05-29 в 15.33.45

Как запустить сервер

Рассмотрим запуск сервера.

Установка необходимых программ

Для начала необходимо скачать архив по ссылке, содержимое которого нужно извлечь на рабочий стол.

https://disk.yandex.ru/d/i9e7Ym_xhbgPyA [flickr:id=52107626890]

Открываем распакованный архив

Запуск

Для того, чтобы запустить локальный сервер, необходимо открыть появившуюся на арбочем столе папку AdventuresInMinecraft и найти в ней пакетный файл StartServer.

В папке находятся два файла StartServer, выберите тот, который подходит вашей операционной системе.

Снимок экрана 2022-05-29 в 15.35.48

После запуска исполнительного файла появится командной окно.

Нажимаем Enter

Теперь серверу нужно прогрузиться

Во время загрузки будут появляться строки о этапах.

Подключаемся к серверу

Теперь, когда сервер запущен, попробуем подключиться к нему.

Адрес сервера 127.0.0.1

Поздравляю, мы установили все необходимое для программирования в Minecraft!
